create

(
  • options
)

Defined in E:\F\myWork\javascript\bootstrap\bt-flyskyhome\js\bootstrap-navbar.js:71

手工显示模态框

Parameters:

  • options Object

    显示方式配置信息

    • create Boolean

      是否创建,是缺省参数,当传入参数非string类型时

    • direction String

      排列方向,v:竖排,h:横排 ,default:v

    • items Array

      列表项，可以和remote同时使用，同时时,该项内容放在remote信息之上

    • remote String

      获取远程列表数据

      • url String
        远程数据获取url
      • param Object
        数据获取参数信息
      • callback Function
        回调函数,远程数据获取成功后执行,通过返回数据对象中的itemList项生成列表

Example:

 //推荐
 $(sSign).navbar(
 {
    items:[{
        text:"我的桌面",
        imgurl:"../Resources/Images/menu/menu_desk.png",
        key:"desktop"
    }],
    remote:{
        url:"../FGQTest/ajax/contentProvide.aspx",
        param:{type:"GetModuleInfo"},
        callback:_initMenuState
    }
  );
 //空列表
 $(sSign).navbar("create");
