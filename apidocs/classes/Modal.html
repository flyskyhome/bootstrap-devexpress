(
  • options
)

Defined in E:\F\myWork\javascript\bootstrap\bt-flyskyhome\js\bootstrap-modal.js:97

手工显示模态框

Parameters:

  • options Object

    显示方式配置信息

    • title String

      标题

    • backdrop Boolean

      是否添加背景元素。=static时，点击模态对话框的外部区域不会将其关闭

    • keyboard Boolean

      按Esc 键时是否关闭模态框

    • show Boolean

      初始化时即显示模态对话框

    • remote String

      如果提供了远程url地址，就会通过 jQuery的load方法加载内容并注入到.modal-body中

    • content String

      模态框内显示的文本内容

    • frame String

      嵌入模态框内的页面地址

    • height String

      模态框内容显示高度

    • width String

      模态框显示宽度

Example:

 //初次显示可用
 $(sSign).modal(options);
 //以后显示可用
 $(sSign).modal("show");
